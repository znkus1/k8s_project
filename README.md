# k8s_project
### 프로젝트 개요
- 항공데이터를 입력시에 지연시간을 예측하는 ML서비스 제공
- 쿠버네티스 클러스터를 MLOps 파이프라인을 구축

### 팀원 및 역할
- 항공 데이터 EDA
- Kubernetes환경에서 Spark 설정
- Pyspark를 이용한 ML코딩
